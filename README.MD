# kiat-md5

[![Greenkeeper badge](https://badges.greenkeeper.io/Hongarc/kiat-md5.svg)](https://greenkeeper.io/)

Simple module to create hash md5

## Installation

````
npm install kiat-md5
or
yarn add kiat-md5
````

## Usage

````javascript
const md5 = require('kiat-md5');
 
console.log(md5('<your_message>'));
````
